Malaysia Covid-19 API

A prototype REST API for Malaysia Covid-19 public data from MOH. Epidemic Ref: MOH-Malaysia Covid-19 Public Data Github.
Vaccination Ref: CITF Malaysia Public Data Github.
Source code: Github Malaysia Covid-19 API.
Format: json
Path: {index, epidemic, vaccine}
Field epidemic: {cases, deaths, tests, hospital}
Field vaccine: {null}
Query: {start_date, end_date, state}

Epidemic usage example:
https://api-covid19-malaysia.herokuapp.com/epidemic/cases?state=Selangor&start_date=2021-01-02&end_date=2021-01-05

Vaccine usage example:
https://api-covid19-malaysia.herokuapp.com/vaccine?state=Selangor&start_date=2021-01-02&end_date=2021-01-05
